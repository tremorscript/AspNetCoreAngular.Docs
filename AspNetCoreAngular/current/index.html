Introduction

This is the developer documentation for the AspNetCoreAngular Starter Kit repository.

What is the point of this repository?

The repository is to be used as the starting point for any project in your organization with Angular as the front-end framework and .Net Core as the back-end.

Features

  • ASP.NET Core 7

  • Entity Framework Core 7

  • Identity Server 4

  • Angular 16

  • Angular CLI

  • AG Grid

  • SASS

  • Secure - with Custom Security Policy (CSP) and custom security headers.

  • Best practices for Angular Code generation.

  • Angular application inspired from asadsahi and gothinkster.

  • Clean Architecture inspired from Jason Taylor and Steve Smith.

  • Unit Testing with Jest.

  • E2E Testing with Playwright.

  • Compodoc for angular documentation.

  • Swagger as the Api explorer.
